## Proyect Configuration

### Important
- This project only works in unix based systems

### Dependences
- Docker
- Docker-compose
- Node.js 20.*.*

### Install the project
- Clone the repository
- ```bash
  $ npm install
  ```

### Start the project
#### Development
- ```bash
  $ npm run docker-dev
  ```

#### Test
- ```bash
  $ npm run docker-test
  ```
